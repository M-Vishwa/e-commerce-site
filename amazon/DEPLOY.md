# Deployment Configuration

This project can be deployed to various platforms.

## GitHub Pages (Web Version)

1. Go to repository Settings > Pages
2. Select source as "main" branch
3. The website will be available at: https://yourusername.github.io/e-commerce-site/

## Render

1. Connect your GitHub repository to Render
2. Set build command: (empty for static site)
3. Set publish directory: amazon/

## Vercel

1. Import the repository to Vercel
2. Set root directory to "amazon"
3. Deploy automatically
